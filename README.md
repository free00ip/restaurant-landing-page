# restaurant-landing-page

A responsive restaurant website built using BEM methodology and SASS.

## 🌐 Live Demo

[View live demo](https://free00ip.github.io/restaurant-landing-page/)

## 📌 About the Project

A single-page restaurant website developed with a focus on responsive design, scalable CSS architecture, and maintainable code structure.

The project uses the BEM methodology for organizing CSS classes and SASS features to create reusable and well-structured styles.

## 🛠 Technologies

- HTML
- CSS
- SASS
- BEM methodology
- Flexbox
- CSS Media Queries
- Responsive Web Design

## ✨ Features

- Fully responsive layout for desktop, tablet, and mobile devices
- BEM-based CSS architecture
- SASS preprocessing with reusable mixins
- Flexible layouts built with Flexbox
- CSS media queries for adaptive design
- Structured and maintainable stylesheet organization
- Semantic HTML markup
